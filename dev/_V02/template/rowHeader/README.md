	images format is:
		#{HeaderImage}_#{lang}.jpg for Header
		and #{CTAImage}_#{lang}.jpg for CTA

4. the text from header is inside **rowHeaderTitle** variable
5. the line of text is inside **row01** variable
6. and copy of CTA is inside **ctaCopy**
